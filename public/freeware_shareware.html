Useful Freeware & Shareware

Here's some information about over 70 freeware and shareware programs which I've found useful for different tasks... Many of these programs were recommended to me by colleagues, and some I stumbled across myself. All of these programs are known to work on Windows 2000 and XP; I have not tested any of them with Vista. Hope that you find some of these useful!
The list is split into 7 sections:
  • Security
  • Backup & recovery
  • System administration
  • Graphics
  • Audio
  • Video
  • File & folder management
  • Communication
  • Internet
  • Educational
  • General

Security

Name Website What it does Notes
Ad-Aware SE Personal http://www.lavasoftusa.com/products/ad-aware_se_personal.php Scans your computer for spyware and security threats Free for non-commercial use
Spybot Search & Destroy http://www.safer-networking.org Scans your computer for spyware and security threats Freeware
Sygate Personal Firewall http://www.majorgeeks.com/download3356.html Free firewall. I like it because it's solid and highly configurable - eg you can customise your own advanced rules according to whatever parameters you want. Freeware. Unfortunately, it is not being developed any more, the last version (5.6.2808) came out in 2004.
TrendMicro Sysclean http://www.trendmicro.com/download/dcs.asp, go to "If you are not a Trend Micro customer" and download the Sysclean package. Scans your computer for viruses and other threats. It can often find and clean up virus residue that antivirus products leave behind. Free for non-commercial use. The TrendMicro site also hosts an online virus scanner and firewall tester.
Windows XP AutoPatcher http://www.autopatcher.com Installs Windows updates without needing an Internet connection - you just install the program, run it, and choose which updates to install. Freeware. Great for updating multiple PCs, or where the Internet is slow or non-existent. Only works on English language versions of Windows. Check the site to see if there is a release for your language as well.

Backup & recovery

Name Website What it does Notes
Cobian backup http://www.educ.umu.se/~cobian/cobianbackup.htm Lets you automate file backups, will back up to local drives and networks. Very configurable, easy to use. Freeware, open source. Nice support forum.
PC Inspector File Recovery http://www.pcinspector.de/file_recovery/UK/welcome.htm Recovers deleted files. I've used this successfully to recover Word documents and other files from a flash drive which was formatted by mistake. Freeware
Zero Assumption Digital Image Recovery http://www.z-a-recovery.com Recovers deleted files. I've used this successfully to recover jpg files from a corrupted CF card. Free version recovers only images, full version recovers other files

System administration

Name Website What it does Notes
Belarc advisor
 http://www.belarc.com/free_download.html
 Makes an html profile of your computer, including hardware, software and security settings
 Free for non-commercial use
Easy Cleaner http://personal.inet.fi/business/toniarts/ecleane.htm Registry cleaner, identifies invalid entries and removes them. Has other cleanup options too, which I haven't tried. Freeware
Everest http://www.majorgeeks.com/download4181.html Like Belarc Advisor, but gives much more detail. Gives all the info you could want to know about your system. Freeware (Free edition no longer in development)
Hamachi https://secure.logmein.com/products/hamachi/default.asp Lets you create a virtual LAN over the Internet, like VPN, but much simpler to set up. See my article about Hamachi on https://itservants.net/Members/richard_young (account needed). Free version allows 16 computers in each network, full version has extra features
LANTool http://www.n37dev.com Remote administration tool for networks. Lets you see at a glance which computers are online, which processes are running. Lets you control many functions remotely. Freeware. Currently in beta.
NetStumbler http://www.netstumbler.com For testing signal strength and more on wireless networks Freeware
NetSwitcher http://www.netswitcher.com Switches between multiple network configurations - useful if you move around different locations. Shareware
NewSID http://www.microsoft.com/technet/sysinternals/Security/NewSid.mspx Changes a computer's "Computer Security Identifier" (SID). Can be used as a less problematic alternative to SysPrep, for cloning systems. Freeware
Process Explorer http://www.microsoft.com/technet/sysinternals/utilities/ProcessExplorer.mspx For each process running on your computer, shows you which files are open, which services, handles and DLLs are being used. Very useful for tracking down OS errors
Registry Editor Extensions
 http://www.dcsoft.com/products/regeditx Replacement for Registry Editor - makes it much more user-friendly. Freeware
Registry Explorer http://www.regxplor.com Turns the Registry into a folder in Windows Explorer. Freeware. A network edition is also available (not free).
TightVNC http://www.tightvnc.com VNC lets you connect to the desktop of a remote machine, and control it. TightVNC gives you various compression and quality settings to minimise bandwidth, so it is great for optimising slow connections. Freeware, multi-platform
Xteq X Setup 6.6 http://www.aplusfreeware.com/categories/LFWV/X-Setup.html Tweak over 1600 Windows settings. Like TweakUI, but much more powerful. 6.6 is the last free version. If the link doesn't work, try a Google search.

Audio

Name Website What it does Notes
AudioGrabber http://www.audiograbber.com-us.net/ Converts between different audio formats, especially good for CD ripping. Lets you rip to MP3 without creating an interim .wav file. Freeware (used to be shareware)
Goldwave http://www.goldwave.com Professional audio editing software, very versatile. Shareware
JetAudio
 http://www.jetaudio.com
 Plays various audio and video files. Can play some streaming Real Player files without Real Player being installed!
 Freeware for the basic (more than adequate) version. If it won't work with Real Player files, try the previous version!
Lame http://lame.sourceforge.net Makes mp3 files from wav files (typically reduces file sizes by 90%) Freeware
WinAmp http://www.winamp.com Audio player for wav, mp3 and other formats. Lets you create playlists, graphic equaliser and other nice features. Freeware

Video

Name Website What it does Notes
AnyDVD http://www.slysoft.com Lets you play DVDs from any region on a region-coded DVD drive Shareware. You have 30 days to try it. It's worked on all the DVDs I've tried it with. Slysoft also produce a nice program called CloneDVD, for copying DVDs.
Subtitle Workshop http://www.urusoft.net Lets you create & edit subtitle files while showing the video in real time Freeware. This is a professional quality program, I use it for quickly adjusting the timing on downloaded subtitles. Urusoft produce an interesting-looking open source media player as well, called ViPlay. 
VLC http://www.videolan.org/vlc Versatile open source media player, plays various formats, including DVDs Freeware, open source

HHD Hex Editor
 http://www.hhdsoftware.com
 For editing binary files
 Freeware version lacks some of the features of the full version
IZArc http://www.izarc.org Archive utility which supports almost all formats, and comes with multilingual user interface Freeware
NoteTab http://www.notetab.com Manipulates and edits text files. Has a built-in macro language to automate tasks. See my article about NoteTab's macro language on https://itservants.net (account needed) - to find it, just search there for NoteTab. Freeware
Powerdesk http://www.v-com.com/product/PowerDesk_Free_Trial.html Windows Explorer replacement, gives you a two-pane view; many useful features. Freeware version lacks some of the features of the full version
Robocopy http://tinyurl.com/4k28b Command-line powerful replacement for XCopy, switches let you configure almost anything Part of Windows Server 2003 Resource Kit Tools. Can also be downloaded on its own if you hunt around. There's also a "Robocopy GUI" available, which I haven't tried.
SplitFile Shell Extension
 Search for it with Google. You can download it from several sites.
 Splits large files, so you can fit them on floppies, CDs, etc. Can make a self-contained .exe file to put the parts back together again. Customisable and very useful.
 Freeware
Synchronise It http://www.grigsoft.com/winsin.htm Synchronises folders between two computers Shareware version is fully functional
Treesize http://www.jam-software.com/freeware/index.shtml Shows how much disk space is being used by different folders Freeware version lacks some of the features of the full version. Several programs are available to do this job - this one has been around for a log while, and I think it is the best.
Wnbrowse http://www.ngthomas.co.uk/wnbrowse.htm For viewing text files. Integrates nicely with Windows Explorer; prevents accidental changes to files. Freeware

Communication

Name Website What it does Notes
Gizmo http://www.gizmoproject.com VOIP client, like Skype but cheaper Freeware. Reported to be better than Skype for low-bandwidth voice chat, as it uses different codecs.
Nokia Unlock http://unlock.nokiafree.org If your  mobile is "locked" to a certain network, this program gives you the code to "unlock" it and use it with any network. Works with Samsung, Siemens, Maxon, NEC, Sony, Panasonic, Vitel, AEG / Telital, Alcatel. Freeware. I bought the link to this program from Ebay. Could have saved myself some money if I'd searched on Google first!
Openfire and Spark http://www.jivesoftware.com Openfire is a Jabber server program, which will let you set up your own server for instant messaging, independently of the Internet. For more information on setting up a Jabber server, look at https://itservants.net/Members/richard_young (account needed).
Spark is one of many Jabber client programs. I like it because it has a simple, clean interface, and is easy to operate for novice users.		 Freeware, multi-platform
Pidgin (used to be called Gaim)
 http://www.pidgin.im
 Like Trillian, but also supports non-Roman scripts (Russian, Japanese, Korean, etc)
 Open-source, and multi-platform
PSI http://psi-im.org PSI is another Jabber client program. It has an unhelpful user interface, and is not that friendly to use. I recommend it only because it lets you log into more than one Jabber account at a time. If you don't need this functionality, stick with Spark. Freeware
Trillian
 http://www.ceruleanstudios.com
 Secure chat client that lets you use MSN, Yahoo Messenger, ICQ, etc, without installing the associated program.
 Freeware version lacks some of the features of the full version.
VOIPBuster http://www.voipbuster.com VOIP client, like Skype but cheaper. Offers free calls to many countries. Freeware
